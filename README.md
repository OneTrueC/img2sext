imgSex
======
Renders images into a set of UTF-8 sextant blocks (black and white
only).

The name *could* have been `imgSextantifer`, but that somehow seems
worse.

Usage
-----

`imgSex FILENAME`

Example
-------

xkcd 3041 as UTF-8 (requires a good monospace font)
